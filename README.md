# CryptoCoinAnalyser
Utility to find dramatic growing up coins

Its not like a real documentation, this is paper for uncompletable pet-project
This programm use Russian Language for menu. Probably I will change it, but when I have time :3

To run programm you should:
1) Get API-key from coinmarketcap https://coinmarketcap.com/academy/article/register-for-coinmarketcap-api
2) Get API-key from Zapper https://build.zapper.xyz/docs/api/
3) Create .env file
4) copy that and past your API-Keys
```
COINMARKETCAP_API_KEY=YOUR_API_KEY
ZAPPER_API_KEY=YOUR_API_KEY
```
